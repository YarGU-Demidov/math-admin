Добро пожаловать на текущую версию административной панели сайта математического факультета. С помощью таблицы ниже вы можете ознакомится с тем, что есть на сайте и для чего.

Пользователи

Пользователи, они же юзеры. Модель взаимодействующая с сайтом на основе прав.

У каждого пользователя должна быть логин, пароль, группа и лицо.

Лица

Лица представляют собой непосредстенно студентов и хранят их личную информацию.

Обязательными параметрами являются: имя, фамилия, номер мобильного телефона.

Преподаватели

Хранят дополнительную информации, касающуюся преподавательского состава.

Категории

Представляют собой различные "теги" по которым будут фильтроваться новости.

Файлы

Возможность загружат/удалять/скачивать различные файлы на сервер.

Для удобства администрирования имеется возможность создавать и удалять директории для файлов.

Директории никак не связаны с файловой структурой сервера. Файлы потребуются в дальнейшем для добавления картинок к новостям,

аватаров для пользователей и других целей.

Вне зависимости от добавленных копий, на сервере файлы храняться только в одном экземпляре.

Использующийся файл невозможно удалить.
